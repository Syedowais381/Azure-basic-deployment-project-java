# Employee Management System

Simple Spring Boot CRUD app for Azure deployment (assignment project).

## Run locally

```bash
mvn spring-boot:run
```

Open http://localhost:8080

## Deploy to Azure

See **DEPLOY.md** for step-by-step Azure Portal instructions.

## Tech stack

- Java 21, Spring Boot 3, Maven
- Thymeleaf (UI)
- H2 (local) / Azure SQL (production)
